# Esercitazione CRUD – Stage Exprivia

## Descrizione del progetto

Questo progetto consiste nella **creazione di un’applicazione CRUD semplice** (Create, Read, Update, Delete), con l’obiettivo di implementare gradualmente nuove funzionalità e strumenti come **Lombok** e **MapStruct**.

L’applicazione consente la **gestione degli utenti**, permettendo di:

- **Registrare** un nuovo utente (con *Id*, *Nome*, *Email*)
- **Visualizzare** la lista completa degli utenti
- **Modificare** i dati di un utente esistente
- **Eliminare** un utente

## Tecnologie utilizzate

- **Java**
- **Spring Boot**
- **Lombok**
- **MapStruct**
- **Maven**
- **Git/GitHub**

## Obiettivo

L’obiettivo di questa esercitazione è comprendere e applicare i concetti fondamentali di:

- Struttura di un progetto Spring Boot
- Mapping tra entità e DTO con *MapStruct*
- Utilizzo delle annotation di *Lombok* per ridurre il boilerplate code
- Gestione base di operazioni CRUD
